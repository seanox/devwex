# Beschreibung

Seanox Devwex ist ein minimalistischer Laufzeit-Container mit einer modularen
Architektur für (Web)Server und Anwendungen. Die enthaltene
Server-Implementierung unterstützt u.a. HTTP mit Virtual Hosting, Filtern,
HTTP-Modulen, SSL/TLS, XCGI/CGI1.1, Directory Listing, Templates und einem auf
Telnet basierenden Fernzugriff zur Steuerung des Laufzeit-Containers. Weitere
Server, Module und Anwendungen lassen sich über die verfügbaren APIs einbinden.
Der Laufzeit-Container und die enthaltenen Server sind reine
Java-Implementierungen und mit entsprechender Laufzeitumgebung auf vielen
Betriebssystem verwendbar.
